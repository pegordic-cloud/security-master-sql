# Security Master SQL Project

A mock investment data operations project built to demonstrate security master 
and pricing exception workflows used in asset management firms.

## What it covers
- Security master table setup across bonds, bank loans, and ABS
- Daily pricing exception reports: missing prices, stale data, coverage gaps
- Portfolio composition summary by asset type

## Tools
SQL (SQLite)

## Context
Built as part of interview preparation for an investment data operations role, 
modeled on real workflows used by alternative investment firms.
